# Cursor Lab San Salvador (UFG)

Una sesión práctica de dos días donde vas a construir aplicaciones completas usando Cursor, React y Supabase. El primer día nos enfocamos en web, el segundo en mobile. No son slides: 70% del tiempo es código en vivo.

Al final del lab, tendrás apps funcionales desplegadas y listas para mostrar. Funciona para cualquier nivel técnico, incluso si estás empezando.

## Qué vas a construir

**Día 1 - Web**: Una aplicación web moderna con React y Supabase que incluye autenticación, base de datos en tiempo real, interfaz responsive y deploy en producción.

**Día 2 - Mobile**: Una aplicación móvil nativa con Expo, conectada a la misma base de datos Supabase, lista para instalar en tu dispositivo.

Todo usando Cursor para acelerar el desarrollo y aprender patrones profesionales en el proceso.

## Para quién es

- Estudiantes de ingeniería o carreras técnicas que quieren practicar con herramientas reales
- Desarrolladores que quieren aprender cómo Cursor mejora su flujo de trabajo
- Cualquier persona con conocimientos básicos de programación que quiera construir apps web

No necesitas ser experto en React o Supabase, vamos paso a paso.

## Requisitos

Antes del evento, asegúrate de tener:

- **Laptop con Cursor instalado** ([cursor.com](https://cursor.com))
- **Cuenta de Cursor** (gratis o Pro)
- **Cuenta de Supabase** (gratis en [supabase.com](https://supabase.com))
- **Node.js instalado** (versión 18 o superior)
- Conocimientos básicos de HTML/JavaScript (no necesitas ser experto)
